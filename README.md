# Docker

Dockerfiles and setup scripts for deploying Docker itself, Portainer, and Zimbra.

## Contents

**docker/**
- `dockerinstall.dockerfile` — Docker Engine installation via Dockerfile
- `install-docker.ps1` — PowerShell script to install Docker on a Windows host

**portainer/**
- `install-portainer.dockerfile` — Deploys a Portainer container management UI instance
- `update-portainer.dockerfile` — Updates an existing Portainer installation

**zimbra/**
- `install-zimbra.dockerfile` — Installs and configures Zimbra Collaboration Suite in a container
